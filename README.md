# SE_Expense_Tracker
軟體工程作業二：費用追蹤工具

## 專案簡介
本專案為一個由兩位學生共同完成的簡易費用追蹤系統，使用者可以輸入多筆費用資料，並依照費用分類產生圓餅圖，以視覺化方式呈現各類支出的比例。
本專案同時示範了使用 Git 分支進行協作開發與整合的流程。

## 功能需求
* 可新增多筆資料 :
  * 日期
  * 金額
  * 分類
  * 備註

* CSV檔案用來儲存費用資料
* 依照分類產生費用比例圓餅圖

## 分工
-feature-input : 費用輸入模組(呂昀靜)
-feature-piechart : 費用視覺化模組(藍昱璇)
